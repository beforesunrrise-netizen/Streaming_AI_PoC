## 🌐 다른 사람들과 공유하기

### 같은 Wi-Fi에서

실행 후 터미널에 표시되는 **Network URL**을 공유하세요:

```
Network URL: http://192.168.0.10:8501
```

이 주소를 다른 사람에게 보내면 같은 Wi-Fi에서 접속 가능합니다!

### 인터넷에서 (무료)

1. GitHub에 코드 업로드
2. [Streamlit Cloud](https://share.streamlit.io) 접속
3. "New app" 클릭 → GitHub 저장소 연결
4. 자동 배포! URL 공유

예시: `https://your-chatbot.streamlit.app`

## 🤖 AI 답변 활성화 (선택)

더 자연스러운 답변을 원한다면:

1. `.env` 파일 생성:
```bash
USE_LLM=true
OPENAI_API_KEY=여기에_API_키_입력
```

2. 다시 실행:
```bash
streamlit run app_chat.py
```

**참고:** API 키 없이도 기본 기능은 모두 작동합니다!

## ❓ 문제 해결

### 포트가 이미 사용 중이라고 나올 때

다른 포트로 실행:
```bash
streamlit run app_chat.py --server.port 8502
```

### 다른 사람이 접속이 안 될 때

1. 같은 Wi-Fi인지 확인
2. 방화벽 확인:
   - Windows: Windows Defender 방화벽 설정에서 8501 포트 허용
   - Mac: 시스템 환경설정 → 보안 및 개인 정보 보호

### 종목을 못 찾을 때

정확한 이름을 사용하세요:
- ✅ "삼성전자" (O)
- ❌ "삼성" (X)
- ✅ "005930" (O)

## 💡 유용한 팁

1. **종목 변경**: 사이드바에서 "종목 초기화"
2. **대화 삭제**: 사이드바에서 "대화 기록 삭제"
3. **모바일**: 휴대폰에서도 완벽하게 작동!
4. **동시 사용**: 각 사용자는 독립적인 대화 유지
