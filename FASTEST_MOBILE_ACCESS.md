## 🚀 대안: ngrok (즉시, 임시)

컴퓨터를 켜둘 수 있다면 ngrok이 가장 빠릅니다.

### 설치 및 실행:

**1단계: ngrok 다운로드**
- https://ngrok.com/download
- Windows 버전 다운로드
- 압축 해제하여 아무 폴더에 저장

**2단계: Streamlit 실행**
```powershell
# 터미널 1
streamlit run app_chat.py
```

**3단계: ngrok 실행**
```powershell
# 터미널 2 (새 터미널 열기)
C:\경로\ngrok.exe http 8501
```

**4단계: 완료!**
- ngrok이 제공하는 URL 복사 (예: `https://abc123.ngrok.io`)
- 핸드폰에서 이 URL로 접속!
